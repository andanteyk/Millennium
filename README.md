# Millennium Assault

ブルアカ二次創作 STG (コード用リポジトリ)

https://andanteyk.github.io/MillenniumAssault


## Installation

1. clone が終わったら Unity 2021.1.3f1 で開いてください。
2. WebGL に Switch Platform してください。
3. Main シーンを開けば実行できるはずです。
