About
--------
<color=#0080ff>[ブルーアーカイブ]</color> のキャラクターを そうさして
たたかう たてスクロール シューティングゲームです。

(マウススクロールで かいてきによめます)


Control
--------
そうさは キーボードを つかいます。

<color=#0080ff>[↑←↓→]</color> いどう

<color=#0080ff>[Z]</color> ショット / メニュー けってい
 てきをこうげきします。むげんにうてます

* ショットを うたないと、
すこしはやく いどうできます

<color=#0080ff>[X]</color> スキル
 てきの たま をけせる スキルをうちます
 はつどうちゅうは むてきです
 ただし かいすうせいげんが あります

* ショットをあてて たおすと ゲージがたまり、
 まんたんになると 1 かい ふえます


Players
--------
<color=#800080>* モモイ *</color>
ようきな ふたごの あね
ひろがる ワイドショット
スキルは こうはんいに たまをばらまく

<color=#008000>* ミドリ *</color>
うちきな ふたごの いもうと
じどうで てきをねらう スナイプショット
たまがこうそくで あてやすい

<color=#0080ff>* アリス *</color>
ゲームがすきな しんぴしょうじょ
レーザーは てき・バリアを かんつうする
スキルは きょだいな レールガン

<color=#800000>* ユズ *</color>
ゲームかいはつぶの ぶちょう
ばくはつする グレネードをうつ
スキルは めんせいあつ ミサイル


Enemies
--------
ミレニアムの せいとたちと たたかいます。
HP が なくなるまえに ステージ 6 を
とっぱできれば クリアです。

<color=#800080>* ユウカ *</color>
せいとかい の かいけい
バリアは どちらのこうげきも はじく

<color=#800080>* アカネ *</color>
C&C の そうじのめいじん
かべにあたると はんしゃする こうげきが とくい

<color=#800080>* カリン *</color>
C&C の しんちょうな スナイパー
はやくてきょだいな たまをはなつ

<color=#800080>* アスナ *</color>
C&C の つかみどころのない エージェント
さきがよめない こうげきをする

<color=#800080>* ネル *</color>
C&C の おこりっぽい リーダー
SMG で そうしゃこうげき する

<color=#800080>* ビナー *</color>
とつぜんあらわれた きょだいなかいぶつ
たおしてしまった C&C のかわりに
たたかうことになった


Items
--------
<color=#ff0000>あかいハート</color> HP がふえます (さいだい 8)
<color=#0080ff>あおいとけい</color> スキルがふえます


Tips
--------
* スキルゲージは 100000 スコアで
たまります

* てきをたおすと スコアが たくさんもらえます

* さいしゅうスコアは
 HP  x500000
 スキル x100000

* ひだんしても すぐにスキルをうてば
ダメージを なかったことにできます
 (ゆうよ 0.25 s)


Special
--------
* [F8] キーで 
こうそくしゅうかい できます?

* Player Select で
 [↑↑↓↓←→←→XZ]
すると デバッグモード になります
(おとがなります)

むてきで スキルも うちほうだい
ですが スコアが こわれます
